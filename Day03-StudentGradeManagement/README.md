# Day 03 - Student Grade Management System

## Overview

A Java OOP-based application that manages student records, calculates total marks, average marks, grades, identifies the class topper, and computes the class average.

---

## Features

- Create Student Objects
- Calculate Total Marks
- Calculate Average Marks
- Assign Grades
- Display Student Details
- Find Class Topper
- Calculate Class Average

---

## Technologies Used

- Java
- Object-Oriented Programming (OOP)
- Constructors
- Methods
- Arrays of Objects
- Enhanced For Loop

---

## Grade Criteria

| Average Marks | Grade |
|--------------|--------|
| 90 and Above | A |
| 75 - 89 | B |
| 60 - 74 | C |
| 50 - 59 | D |
| Below 50 | F |
