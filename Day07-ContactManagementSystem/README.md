# Day 07 - Contact Management System

## Overview

A Java console application to manage personal contacts using Object-Oriented Programming. Users can add, search, update, delete, and display contacts through a menu-driven interface.

---

## Features

- Add Contact
- View All Contacts
- Search Contact by ID
- Update Phone Number
- Delete Contact
- Menu-Driven Application

---

## Technologies Used

- Java
- Object-Oriented Programming (OOP)
- Arrays
- Scanner Class
- Constructors
- Methods

---

## Concepts Practiced

- Classes and Objects
- CRUD Operations
- Arrays of Objects
- Searching
- Updating Data
- Deleting Records
- Menu-Driven Programming
