# Day 05 - Bank Account Management System

## Overview

A Java console application that simulates a simple banking system using Object-Oriented Programming principles. Users can create bank accounts, deposit money, withdraw funds, search for accounts, and view all account details.

---

## Features

- Create Bank Account
- View All Accounts
- Deposit Money
- Withdraw Money
- Search Account by Account Number
- Menu-Driven Console Application

---

## Technologies Used

- Java
- Object-Oriented Programming (OOP)
- Arrays
- Constructors
- Methods
- Scanner Class

---

## Concepts Practiced

- Class and Object
- Constructor
- Arrays of Objects
- Menu-Driven Programming
- Deposit and Withdrawal Logic
- Searching
- Basic Banking Operations
