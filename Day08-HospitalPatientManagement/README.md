# Day 08 - Hospital Patient Management System

## Overview

A Java console application to manage hospital patient records using Object-Oriented Programming principles. Users can add, search, update, discharge, and view patients through a menu-driven interface.

---

## Features

- Add Patient
- View All Patients
- Search Patient by ID
- Update Disease
- Discharge Patient
- Menu-Driven Console Application

---

## Technologies Used

- Java
- Object-Oriented Programming (OOP)
- Arrays
- Scanner Class
- Constructors
- Methods

---

## Concepts Practiced

- CRUD Operations
- Arrays of Objects
- Searching
- Updating Records
- Deleting Records
